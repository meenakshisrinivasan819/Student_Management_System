# 🎓 Student Management System (Java)

## 📌 Overview

Student Management System is a console-based Java application designed to manage student records efficiently. The system allows administrators to add, view, search, update, and delete student information while also tracking academic performance and attendance.

This project demonstrates the use of Object-Oriented Programming (OOP), Java Collections Framework, and basic data management concepts.

---

## ✨ Features

* 🔐 Admin Login Authentication
* ➕ Add New Student Records
* 📋 View All Students
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student Records
* 📊 Automatic Percentage Calculation
* 🏆 Grade Generation (A, B, C, D, F)
* 📅 Attendance Percentage Calculation
* 🔄 Sort Students by Name
* 📈 Sort Students by Percentage
* 📌 Dashboard Analytics

  * Total Students
  * Average Percentage
  * Top Performer
